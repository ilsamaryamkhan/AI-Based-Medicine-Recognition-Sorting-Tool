# AI-Based Medicine Recognition & Sorting Tool

An intelligent, full-stack computer vision application designed to automate the detection, classification, and inventory sorting of pharmaceutical products. This project combines a high-performance deep learning pipeline with a responsive web interface to streamline medicine management.

---

## 🚀 Key Features

* **Computer Vision Pipeline:** Real-time object detection and classification of medicines from imagery using YOLO and OpenCV.
* **Custom Deep Learning Models:** Fine-tuned models trained on dedicated, annotated pharmaceutical datasets using TensorFlow and PyTorch.
* **Full-Stack Architecture:** A dynamic, user-friendly React frontend coupled with a robust, lightweight Flask backend REST API.
* **Relational Data Storage:** Structured inventory tracking, medicine metadata management, and logging handled via a PostgreSQL database.

---

## 🛠️ Tech Stack

### Artificial Intelligence & Vision
* **Languages:** Python
* **Frameworks:** PyTorch, TensorFlow
* **Libraries:** OpenCV, Ultralytics (YOLO)

### Full-Stack Web Development
* **Frontend:** React.js, HTML5, CSS3, JavaScript
* **Backend:** Flask (Python)
* **Database:** PostgreSQL
* **API Protocol:** RESTful HTTP
